# Httptester
A basic http stress tester

```
Usage of httptester:
Arguments:
  URL   the destination URL to test
  -X string
        http method to use (default "GET")
  -content-type string
        content-type (default "application/json")
  -d string
        data (body) sent with a POST request
  -n int
        count of test requests to make (default 5)
  -no-spread
        disable randomly spreading requests within one second per repetition
  -reps int
        repetitions of the test to make (default 1)
```
